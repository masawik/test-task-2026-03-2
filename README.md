# Fast start

```bash
git clone https://github.com/masawik/test-task-2026-03-2.git
```

```bash
cd test-task-2026-03-2
```

```bash
cp .env.example .env
```

> Or create `.env` manually using the same content as `.env.example`

docker:

```bash
docker compose up
```

or locally:

```bash
pnpm install
pnpm dev
```
